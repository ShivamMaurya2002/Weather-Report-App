## 🚀 Features

✅ Real-time weather updates  
✅ Search by any **city, village, district, or country**  
✅ 🔍 Geolocation with **OpenWeatherMap Geocoding API**  
✅ 📅 5-day weather forecast with temperature, icons & date  
✅ 🌡️ Detailed data: temperature, feels-like, humidity, pressure, wind  
✅ 🎨 Clean UI with **professional design & animations**  
✅ 📱 Fully **responsive** on all devices  
✅ 🔄 Dynamic data rendering without page reload
